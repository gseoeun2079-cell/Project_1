github은 코드 작성과 코드 수정을 한 파일을 저장하고 다른 사람과 협업할 수 있는 사이트이고, streamlit cloud은 github과 연계되어 파일을 쉽게 배포하는 사이트이다.

배포는 제가 만든 코드의 파일을 다른 사람이 볼 수 있거나 사용할 수 있게 하는 것이다. 하지만 배포하여 코드를 사용하는 사용자는 코드를 수정할 수 없다.

1) 컴퓨터에서 코드 작성
2) github에 저장과 다른 사람들과의 협업
   1) .py은 실행하고 싶은 코드 파일
   2) requirements.txt는 가져오고 싶은 파일을 나열하는 파일(pip install의 역할을 하는 파일)
3) github과 연결되어 있는 streamlit cloud에서 간단하게 배포
